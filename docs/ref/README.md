# Introduction

This section provides reference documentation for AssetGuard developers and operators. The `ref/` hierarchy contains detailed descriptions of core modules, configuration options, APIs, and deployment processes.

Use this section as your go-to reference when you need to understand how a specific subsystem works, how it is configured, or where to locate the code that implements it.

Contents:

- **Getting Started**: Requirements, installation, and package overview
- **Configuration**: How to configure agent and manager components
- **Modules**: Full reference for major subsystems (FIM, SCA, Syscollector, etc.)
- **Upgrade/Backup/Restore**: Operational workflows for managing the platform
- **Security & Performance**: Best practices and hardening guidance
- **Glossary**: Common terminology used throughout the project

For onboarding and quick context, see [Developer Onboarding](ref/onboarding.md).
